CardTable

CardTable (for lack of a better name) provides a virtual table top that you can play any two player card game on. A standard deck of 52 playing cards is provided, but you can create any deck you want. CardTable was written with SW:CCG and ST:CCG in mind, but there is nothing in the program itself that implements any of the game mechanics. What it does provide is standard card actions: Shuffle, Pick up, Move, Fan, etc.

This program is provided “as is”, with no guarantee of suitability, performance or support. DeckTech does not provide technical support for CardTable or any other software you download from this site.

CardTable is written in pure Tcl/Tk. What that means to you is: Platform independence, and the source is always available. You need Tcl/Tk 8.0p2 for your platform. The Windows 95 and Windows NT self-extracting installer (tcl805.exe) 1.75 Mbytes The file is a self-extracting executable. It will install the Tcl and Tk libraries, the wish and tclsh programs, and documentation.
